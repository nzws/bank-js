# bank-js

> Node.js 向けオンラインバンキングのスクレイピングモジュール

## 対応状況

| 銀行     | 残高 | 履歴 | 詳細な履歴 | 振込 | 参考                                          |
| -------- | :--: | :--: | :--------: | :--: | --------------------------------------------- |
| ゆうちょ | ✔    | ✔    | ⏳         | ❌   | 履歴: 直近 10 日分                            |
| 楽天     | ✔    | ✔    | ⏳         | ❌   | 履歴: 最新 50 件, ゆうちょ → 楽天の自動払込可 |

- このモジュールでは振込の実装をする予定はありません（怖いので）
- 履歴参照は最近のデータのみ対応
